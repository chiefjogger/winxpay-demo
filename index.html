03Merchant integration · the seamless POS layer

One drop-in widget. 1.4M tạp hóa eligible.

Merchants already on the WinX Tạp Hóa POS get WinXPay automatically — it's a payment method in their existing checkout. For everyone else, integration is a single embed: a button, an API call, or a webhook. Same MDR, same brand-funded discount engine, same TCB BNPL leg. No bespoke contracts; no card-terminal estate.

⚡
Drop-in checkout button
One HTML attribute or one SDK call. Renders the WinXPay QR + Pay-in-4 selector inside any web, app, or POS surface. Median integration time: 2 hours.
🔁
Webhook-driven settlement
Real-time payment.captured + bnpl.approved events. T+0 settlement to merchant's WintelPay wallet.
🧾
E-invoice baked in (NĐ 70/2025)
HĐĐT-MTT issued automatically with every settled transaction. Compliance-ready for GDT submission. Tạp hóa never touches a tax form.
🏦
Capital + decisioning, abstracted
Merchant sees one MDR. Behind it: Masan brand budget (CAC), Techcombank capital (BNPL), WinXPay (decisioning + servicing). The merchant takes none of the credit risk.
📈
AOV uplift, not just acceptance
Klarna mature-market data: 30–50% AOV lift on BNPL-eligible transactions. Affirm reports similar. The 4.5% MDR is paid for by the basket the consumer wouldn't otherwise have built.
checkout.html · drop-in widget
<!-- Add the WinXPay button to any HTML page --> <script src="https://js.winxpay.vn/v1/sdk.js"></script> <button data-winxpay="checkout" data-amount="200000" data-currency="VND" data-merchant="MS001247" data-bnpl="true" data-discount-id="MCH-2026-FMCG-Q2"> Trả 200K hoặc 4 × 50K </button>
server.ts · capture + webhook
import { WinXPay } from '@winxpay/sdk'; const wxp = new WinXPay({ apiKey: process.env.WXP_KEY }); // Create a checkout — receive QR + paymentId const checkout = await wxp.checkouts.create({ amount: 200_000, basket: cart.items, bnpl: 'auto', // auto-decide via TCB brandFundedDiscount: true, // pulls MCH budget callbackUrl: 'https://store.vn/wxp' }); // Webhook handler — fires when consumer scans + TCB approves app.post('/wxp', (req) => { if (req.body.event === 'payment.captured') { orders.markPaid(req.body.merchantRef); // VND 200K instant in WintelPay merchant wallet (less 4.5% MDR) } });
